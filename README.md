# 天文台预约 & 船票预订

## 项目说明

本项目包含两个独立的自动化脚本，分别用于深圳天文台预约和船票自动预订。

## 文件说明

### main.py - 深圳天文台自动预约

支持三种预约模式：

1. **立即预约** - 手动选择日期和时段立即预约
2. **定点抢号** - 在每周指定时间（如周一10:00）自动抢号
3. **捡漏模式** - 自动检测是否有空位放出并自动预约

**依赖：**
```
pip install requests ddddocr
```

**配置说明：**
- 修改 `wxuser` 为你的微信用户标识
- 修改 `user_info` 列表中的访客信息（姓名、身份证号、手机号）
- 需要准备 `test.png` 用于 OCR 模型初始化测试

---

### ship.py - 鹏星会船票自动查询预订

自动检测**南澳↔盐田**船票余票并自动下单，支持通过 PushPlus 发送通知。

**依赖：**
```
pip install requests
```

**配置说明：**
- 修改 `token` 为你的 PushPlus token
- 修改 `contactId` 为联系人ID
- 可配置目标日期、船班、票种等筛选条件

---

## 使用方法

### 运行天文台预约
```bash
python main.py
```
按提示选择功能模式。

### 运行船票预订
```bash
python ship.py
```
按提示输入 token 和查询条件。
